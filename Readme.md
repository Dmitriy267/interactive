# Интерактивные примеры (картинки, галереи, слайдеры )

## В приложении реализованы следующие примеры:

- Выбор фото

- Простое увеличение фото
- Увеличение непрозрачности
- Выбор из списка
- Галереи
- Смещаем фото вверх
- Фото по номерам
- Перемотка от центра
- Галерея-аккардион
- Галерея с ограничителями
- Замкнутая галерея
- Оптимальная галерея
- Автоматические слайдеры
- Левый слайдер
- Слайдер с растворением
- Добавляем снимки
- Перемещение изображений
- Направляем ракету
- Перемещаем солнце
- Освещаем пейзаж
- Тормозим шарики
- Эффекты
- Пасьянс из картинок
- Раскрашиваем рисунки
- Настраиваем фото
- Проявляем картинку

### Реализация

Данные примеры реализованы с использованием js
